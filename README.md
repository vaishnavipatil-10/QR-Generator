# ⚡ QR Forge — Advanced QR Code Generator

A modern, high-performance **QR Code Generator** built using **Flask (Python)** with a sleek, futuristic UI.

Generate QR codes instantly for any URL or text and download them in PNG format.

---

## 🚀 Features

* ⚡ Instant QR code generation
* 🎨 Premium futuristic UI (glass + neon style)
* ⌨️ Supports both URL & text input
* ⏎ Enter key support for quick generation
* 📥 Download QR code as PNG
* 🔄 Loading animation & smooth transitions
* 🧠 Error handling for better UX

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Frontend:** HTML, CSS, JavaScript
* **Library:** qrcode (Python)

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/qr-forge.git
cd qr-forge
```

### 2. Create virtual environment

```bash
python -m venv venv
```

### 3. Activate environment

**Windows:**

```bash
venv\Scripts\activate
```

**Mac/Linux:**

```bash
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install flask qrcode pillow
```

---

## ▶️ Run the Application

```bash
python qr_genrator.py
```

Open in browser:

```
http://127.0.0.1:5000
```

---

## 📸 How It Works

1. Enter a URL or text
2. Click **Generate** (or press Enter)
3. QR code appears instantly
4. Download with one click
